## 十、常见问题（工具相关）

### Q1: 没有海外信用卡怎么用 OpenAI？

**解决方案**：
- 用 DeepSeek / 通义千问（国内可直接用）
- 找代充（注意风险）
- 用 OpenAI 第三方中转（注意选择可靠的）

### Q2: Docker 在国内拉镜像慢怎么办？

**解决方案**：
```json
// ~/.docker/daemon.json
{
  "registry-mirrors": [
    "https://docker.mirrors.ustc.edu.cn",
    "https://hub-mirror.c.163.com"
  ]
}
```

### Q3: K8s 太复杂，可以跳过吗？

**建议**：
- 如果目标是找工作：建议学
- 如果目标是快速出活：用 Docker Swarm 或单机 K3s
- 本培养计划：基础版用 Docker Compose，挑战版用 K8s

### Q4: 怎么追踪 AI 领域的最新进展？

**推荐方式**：
- Twitter：关注 AI 领域大 V
- Reddit：r/MachineLearning, r/LocalLLaMA
- arXiv-sanity：每日新论文
- Newsletter：TLDR AI

### Q5: 学习 Agent 一定要读论文吗？

**建议**：
- 应用层：不一定（先动手做）
- 研究层：必须读
- 本培养计划：以实践为主，论文作为参考
