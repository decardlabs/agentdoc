## Who is this for?

Software engineers who are comfortable with Python and want to go from "curious about AI agents" to "shipping agents in production". No prior ML experience is required — you just need to be able to read and run Python code.

## How to use these docs

Read the documents in order if you are new to agents. If you already understand the basics, jump straight to the topic you need. Each document ends with a **Further Reading** section pointing to additional resources.
